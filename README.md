# Archive-Fund-Android

Урезанная Android-версия приложения разработано для работы с архивом учебной организации в рамках выполнения учебной практики.
Приложение предназначено для работы с архивом образовательной организации. В нем реализованы функции добавление, редактирование, удаление личных дел.
Приложение обеспечивает хранение информации о личных делах студентов.
С благодарностью принимаю предложения по улучшению и замечания относительно представленного материала.

Приложение «Архивный фонд» разработано для того, чтобы помочь с упорядочиванием повседневных задач. В современном ритме жизни легко упустить важные дела из виду, особенно когда они разбросаны по разным источникам-от бумажных записок до заметок в телефоне. Это приложение объединяет всё необходимое для ведения личных дел в одном удобном месте.
С помощью «Архивный фонд» есть возможность заносить личные дела и просматривать их нахождение в архиве организации. Удобная структура позволяет сортировать дела по сроку отчисления студентов, ФИО и специальности. 
В этом руководстве рассмотрены такие темы как начало работы с приложением, как создавать и редактировать личное дело, удаление личного дела, а также реализация поиска. Простой интерфейс и логичная навигация позволят быстро освоиться.
Начало работы с приложением
При запуске приложения пользователю представляется главный экран с эргономичным пользовательским интерфейсом, оптимизированным для эффективного взаимодействия с ключевыми функциями системы. Элементы управления размещены в логически обоснованной последовательности, что способствует быстрому освоению функционала приложения и минимизации временных затрат на его использование.

[![image.png](https://i.postimg.cc/JnN10rLq/image.png)](https://postimg.cc/xJdrFSqJ)
 
Главная активность

Для добавления нового личного дела пользователь должен нажать кнопку «Добавить личное дело», расположенную в нижней части окна.
После этого откроется форма ввода, где необходимо заполнить все обязательные поля, отмеченные символом (*).

[![image.png](https://i.postimg.cc/xC22Wwzk/image.png)](https://postimg.cc/yJvtRrnB)
 
Добавление нового личного дела

После этого откроется форма ввода данных, где пользователь сможет указать параметры нового дела: фамилию, имя, отчество, форму обучения и другие важные сведения.
Обязательные поля выделены красной звездочкой (*), и при попытке сохранить форму без их заполнения система выдаст предупреждение с указанием на недостающие данные.
Кроме того, интерфейс формы адаптирован под различные размеры экранов, что обеспечивает комфортную работу на устройствах с разным разрешением.

[![image.png](https://i.postimg.cc/wTvp8c70/image.png)](https://postimg.cc/14LTDF0q)
 
Активность для внесения данных о новом личном деле

После ввода данных нужно нажать кнопку «Сохранить».
Для просмотра деталей личного дела достаточно кликнуть по нужной записи в списке — откроется окно с полной информацией и возможностью редактирования или удаления.

[![image.png](https://i.postimg.cc/QMpLkfyx/image.png)](https://postimg.cc/DWyY7Pht)
 
Действия для просмотра подробной информации о деле

Если необходимо отредактировать данные личного дела, необходимо кликнуть по личному делу подлежащему внесению изменений в после списка личных дел.

 [![image.png](https://i.postimg.cc/J0YS4vGG/image.png)](https://postimg.cc/K3L00972)

Действия для редактирования данных 

При нажатии кнопки «Редактировать данные» открывается форма для внесения изменений в ранее сохранённую информацию.
После редактирования, при попытке сохранить данные, появляется всплывающее окно с запросом подтверждения действий перед окончательным сохранением.

 [![image.png](https://i.postimg.cc/ZKg1kkkn/image.png)](https://postimg.cc/Xpf2FzY6)

Подтверждение действий по редактированию данных

Если необходимо удалить данные о личном деле, пользователь должен кликнуть на кнопку «Удалить личное дело» на рисунке 87, расположенную в окне просмотра подробной информации.

 [![image.png](https://i.postimg.cc/KzJHJwrm/image.png)](https://postimg.cc/yg3jNfbG)

Действия для удаления данных

После этого появится диалоговое окно с запросом подтверждения удаления, чтобы избежать случайного удаления важной информации.

 [![image.png](https://i.postimg.cc/TPc73FsR/image.png)](https://postimg.cc/p9pY0Gx7)

 Подтверждение действий по удалению данных

Для поиска личных дел студентов по году отчисления, фамилии, имени, отчеству или наименованию специальности нужно ввести данные в соответствующие поля в верхней части интерфейса.
После ввода параметров достаточно нажать кнопку «Поиск», и на экране отобразятся результаты, соответствующие заданным критериям.

 [![image.png](https://i.postimg.cc/QdG0kjG5/image.png)](https://postimg.cc/RNd7v50V)

Реализация поиска личных дел

Приложение «Архивный фонд», являясь специализированным программным решением, представляет собой высокоэффективный инструмент для организации и систематизации информации, связанной с личными делами студентов. 
Интуитивно понятный интерфейс и тщательно проработанная структура приложения обеспечивают пользователям возможность с высокой степенью эффективности осуществлять операции по добавлению, редактированию, удалению и поиску необходимых записей. Руководство пользователя, представленное в виде детального и исчерпывающего документа, охватывает все ключевые функции приложения, что позволяет даже новичкам быстро освоить основные аспекты его использования.
Программа «Архивный фонд» была разработана с учётом технического задания, для повышения эффективности управления архивными записями и оптимизации документооборота.
